Empowered Essence

For this design project I was tasked with completing six different unique title concepts using only three font variations and utilizing different letterforms and composition for each. I started by sketching out two different ideas for each using the fonts that complemented each concept the best. I really enjoyed this project and shows strength in my design skills by using minimal tools to portray each of these unique typography concepts. Adobe Illustrator was used to create my final design for this project.
